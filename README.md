# 🚀 DSA CodeLab — Live C Compiler

A powerful in-browser C compiler and DSA playground built using HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://cprogramcompiler.netlify.app/

---

## ✨ Features

* 🧠 16 Data Structures & Algorithms programs
* ⚡ Live C code execution using Judge0 API
* 📝 Editable code playground
* ▶️ Run code directly in browser
* 📚 Clean UI for learning DSA
* 💻 No installation required

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* Judge0 API (C compiler backend)
